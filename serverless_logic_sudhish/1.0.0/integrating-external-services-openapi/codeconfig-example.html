Code/Config Example: Deploying with the Dev Profile

Configuration

To deploy a Quarkus application using the Dev profile on OpenShift Serverless, you need to create a kn workflow. Here’s an example of a simple kn workflow YAML file (dev-profile.yaml):

apiVersion: knative.dev/v1
kind: Service
metadata:
  name: my-quarkus-app
spec:
  template:
    spec:
      containers:
      - image: <your-quarkus-app-image>
        ports:
        - containerPort: 8080

Replace <your-quarkus-app-image> with the image of your Quarkus application.
